## 第一步：制作擴充功能的安裝檔 `manifest.json`
每個套件都會擁一個 [manifest](https://developer.chrome.com/docs/extensions/mv3/getstarted/#manifest)，主要提供套件內容、版本、說明、權限等重要資訊，我們可以簡單說就是告訴Google這個extensions要做什麼。

### 官方提供起手式
```json
{
  "name": "Getting Started Example",
  "description": "Build an Extension!",
  "version": "1.0",
  "manifest_version": 3
}
```

### 照著修改一下

```json
{
  "name": "簡單測試",
  "description": "套件的說明",
  "version": "1.0",
  "manifest_version": 3,
  "background": {
    "service_worker": "./js/background.js"
  },
  "content_scripts": [
    {
      "matches": [
        "*://www.google.com/"
      ],
      "js": [
        "./js/content.js"
      ],
      "run_at": "document_idle"
    }
  ],
  "permissions": [
    "tabs"
  ],
  "action": {
    //套件的行為
  }
}
```

*  **name** - 套件名稱
*  **description** - 套件的說明
*  **version** - 在擴充店上架時才會用到的，每次一次更新都需要變化數字來確認版本
*  **manifest_version** - 現在最新版本為V3
*  **background** - 躲在幕後，只做重要事情的頭頭
*  **content_scripts** - 頭頭的小弟，也是跟頁面的主要聯絡窗口
*  **permissions** - 要啟用的權限
